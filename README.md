##  عن المشروع (About)
**المعلم الذكي** هي منصة تعليمية مبتكرة تهدف إلى تحويل الكتب المدرسية والمناهج التقليدية إلى تجارب تعليمية تفاعلية ممتعة باستخدام الذكاء الاصطناعي.

**The Smart Teacher** is an AI-powered educational platform that gamifies learning curricula. It empowers teachers with automated lesson planning tools and engages students through interactive "learning maps" and AI tutoring.

---

##  الميزات الرئيسية (Key Features)

### ‍ للمعلم (For Teachers)
- **مخطط الدروس الذكي:** توليد خطط درس كاملة متوافقة مع المعايير التربوية بضغطة زر.
- **تحليل المناهج:** رفع ملفات PDF واستخراج الوحدات والدروس تلقائياً.
- **لوحة تحكم تفاعلية:** أدوات لإدارة الفصل، التصويت الفوري، وإرسال الملاحظات للطلاب.
- **توليد المحتوى:** إنشاء اختبارات، بطاقات استذكار، وألعاب تعليمية فورية.

### ‍ للطالب - واحة التلميذ (Student Oasis)
- **خريطة التعلم (Learning Map):** تحويل المنهج إلى خريطة كنز تفاعلية.
- **عليم الصغير (Little Aleem):** مساعد ذكي للإجابة على الأسئلة وشرح المفاهيم.
- **نظام المكافآت:** جمع نقاط XP وأوسمة عند إتمام الدروس.
- **مشغل الدروس:** شرائح تفاعلية تعرض المحتوى بشكل جذاب.

##  التقنيات المستخدمة (Tech Stack)
- **Frontend:** React, Vite, Tailwind CSS, Lucide React
- **AI Integration:** Google Gemini 2.0 Flash & Pro
- **Deployment:** Vercel

##  التشغيل محلياً (Run Locally)

1. **استنساخ المستودع (Clone):**
   `ash
   git clone https://github.com/Physiker80/the-smart-teacher.git
   cd the-smart-teacher
   `

2. **تثبيت الحزم (Install Dependencies):**
   `ash
   npm install
   `

3. **إعداد المتغيرات البيئية (Environment Variables):**
   قم إنشاء ملف \.env.local\ وأضف مفتاح Gemini API:
   `env
   VITE_GEMINI_API_KEY=your_api_key_here
   `

4. **تشغيل التطبيق (Run):**
   `ash
   npm run dev
   `
