# the-smart-teacher
The Smart Teacher: An AI-powered educational platform that gamifies learning curricula. It empowers teachers with automated lesson planning tools and engages students through interactive "learning maps" and AI tutoring, transforming static textbooks into dynamic educational adventures.

## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   npm install
2. Set the GEMINI_API_KEY in .env.local to your Gemini API key
3. Run the app:
   npm run dev
